How to use Kittyauth

Add login button

Add an <a href="https://kotelek.dev/kotcli/web/?login&next=https://example.com"> button to login

The https://example.com have to listen for ?access_token GET parameter.

When it receive it has to do CURL GET request to this api: https://kotelek.dev/kotcli/web/api/?get=access_token&toget=userdata

with argument &access_token set to the parameter

When the access_token is correct server will return this json: {"user_email":"example@example.com","user_username":"example"}

When it's incorrect server will return this: {"error":true,"message":"Invalid access token"}

To logout make simple button <a href="https://kotelek.dev/kotcli/web/?logmeout&next=https://example.com/?logged_out">

When server receives ?logged_out it will set session data for example LOGGED to false and not display login-required things!
